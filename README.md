# Earth Lens

Decentralized AI-verified hazard reporting PWA.

## Setup

1. `firebase login`
2. `firebase init` (Hosting + Functions)
3. Replace placeholders (Firebase config, API keys)
4. Set Gemini key: `firebase functions:config:set gemini.api_key="YOUR_GEMINI_KEY"`
5. `firebase deploy`

Works offline, installable, real-time map.
